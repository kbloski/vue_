Streszczenie:

v-if, v-else-if, v-else - muszą występować w elementach html zaraz po sobie ponieważ v-else-if, v-else czerpią informacje z sąsiedzniego elementu, ta funkcja ingeruje w dom, ona naprawdę usuwa i dodaje elementy do drzewa dom

v-show='warunek' - ukrywa elementy gdy warunek nie jest spełniony i pokazuje gdy jest spełniony 

                    v-for="(goal,index) of goals" // no use

                    v-for="(goal,index) in goals" - po listach
                    v-for="(value,key,index) in user" - po obiektach
                    v-for="num in 5" - po liczbach

v-bind:key="goal" - dodaje unikalność dla każdego elementu generowanego za pomocą v-for, dzięki czemu taki element jak input zachowuje swoje miejsce nawet w sytuacji, gdy usuwamy element poprzedni
